# URL-shortener
Simple URL shortener based on Nodejs and MySQL
Created by JunYoung Lee


This code usees MySQL to save data(id, long url, short url).

Uses 62 letters to encode data's id number.

Used NPMs
- express
- jade
- node-mysql
- request
- request-promise
